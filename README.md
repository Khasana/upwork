## To set up APi
command:
```
1. cd api
2. npm install
3. npx drizzle-kit push
4. npm run dev 

open http://localhost:8989
```
## frontend
```
1. Open new terminal window
2. cd web 
3. npm install
4. npm run dev

Open the link http://localhost:5173 in your browser

http://localhost:5173/app/tasks - use this link to see list of tasks and create new task.

```
